⚠ 不審な session
運営者・作家自身のテストいいねの可能性がある session を自動検出。 選択して 除外する と、その session の全 likes が集計から除外されます (再度押せば解除)。 除外操作は Cloud Function 経由で記録されます。
🔍 自動検出されない session を手動で除外:
sessionId はブラウザの localStorage に保存されている識別子。自分のスマホの localStorage を覗けば確認できます。
🎭 作家別 (グループ展)
グループ展では、作品ごとではなく 作家単位 での評価が重要。 リーチ (= 興味を持った visitor の人数) と平均深さ (= 1 visitor が押した数の平均) を併記。 単一の「人気作」よりも、作家への幅広い関心を捉える。
