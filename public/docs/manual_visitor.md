# 来場者向けマニュアル

会場で気に入った作品に「いいね」したり、感想を送ったりする手順、および Web 展覧会 (オンラインで見られるバージョン) の使い方をまとめます。スマートフォンがあれば誰でも参加できます。専用アプリのインストールは必要ありません。

---

## A. 会場で QR コードを読み取って参加する

### A-1. 作品の横にある QR コードを読み取る

会場では、作品の近く (キャプションの隅やパネルの下など) に小さな QR コードが貼ってあります。
スマホのカメラでその QR コードを読み取ると、その作品の専用ページが開きます。

### A-2. 作品ページが開きます

ブラウザ (Safari や Chrome) が立ち上がり、作品のページが表示されます。

ページに表示されるもの:
- 作品の写真
- 作品タイトル
- 作家の名前
- (作家が登録していれば) 作家の SNS リンク (Instagram、X、Web サイト等のアイコン)
- 感想入力フォーム

### A-3. 感想を送る

ページを下にスクロールすると、フォームが出てきます。

1. **ニックネーム** の欄: 表示名を入れます (空欄のままでも OK。空欄の場合「(匿名)」として記録されます)
2. **感想** の欄: 作品を見た感想を文章で書きます (一言でも長文でも自由)
3. 一番下の **「感想を送って『いいね』」** ボタンを押します

送信に少し時間がかかります (1〜2 秒)。

成功すると、ハートマークと一緒に「**有難うございました！ 応援数: ○**」というお礼メッセージが出ます。

### A-4. 送信を間違えた / 取り消したい

送信完了画面の下に **「🔄 取消」** ボタンがあります。

- これを押すと**直前に送信したいいね・コメント** が取り消されます
- ボタンを押せるのは送信直後のページが開いている間だけ
- ページを閉じたり別の作品に移動した後は、取消できません (主催者にご連絡ください)

### A-5. 同じ作品にもう一度コメントしたい

一度送信した後、もう一度感想を書きたくなったら:

- 2 回目以降のボタン文言は「**追加の感想を送る**」になります
- いいねカウントは 1 人 1 票で固定 (同じスマホからのいいねは何度押しても 1 票) ですが、感想は何件でも追加できます

### A-6. 「いいね受付済みです、追加コメントを受け付けます。」が出た場合

過去にこの作品にいいねしたことがあるという意味です。
そのまま追加の感想を書いて送ることができます。

### A-7. 「現在は受付期間外です」が出た場合

展覧会によっては「会期中だけ感想受付」のような期間設定をしています。
期間外の場合、感想フォームの代わりに黄色いお知らせが表示されます。
この場合、感想は送れません。期間が始まったら再度試してください。

### A-8. 練習用展覧会の表示

ページ上部に「🧪 これは練習用展覧会です」というバナーが出る場合は、主催者がテストとして用意した展覧会です。あなたのコメントは 14 日後に削除されます。

## B. Web 展覧会で作品を見る

会期が終わった後の常設アーカイブとして、また会場に来られない方向けに、Web 上で作品を一覧できる **Web 展覧会 (バーチャル展覧会)** を公開している主催者もいます。

### B-1. アクセス方法

主催者から告知された URL を開きます。形式は次のいずれか:

- **公開タイプ**: 普通の URL (主催者が SNS や Web サイトに貼っているもの)
- **招待タイプ**: 主催者から QR や DM 等で個別に配布された長めの URL

どちらもブラウザで開けば作品の一覧 (グリッド) が並んで表示されます。

### B-2. 作品をタップして詳細を見る

各作品をタップすると、大きい画像と詳細情報がモーダルで開きます。

- 画像をピンチ操作で拡大可能
- スワイプで前後の作品に移動できる場合もあります
- 作家にショップやウェブサイトがあれば「ショップを見る →」のリンクが出ます

### B-3. いいねする / 取り消す

主催者が「いいね」を有効にしている場合のみ表示されます。

1. モーダル内の **♡ アイコン** をタップ → ハートが ♥ (赤) に変わり、いいねが記録される
2. もう一度 **♥ アイコン** をタップ → 取り消し

いいね数はリアルタイムで他の来場者にも反映されます。

### B-4. コメントを書く / 削除する

主催者が「コメント」も有効にしている場合のみ。

1. モーダル内の **💬 アイコン** をタップ → コメント欄が下に開く
2. ニックネーム (任意) と感想を入力 → 「送信」
3. 送信したコメントは下のリストにすぐ表示される
4. **自分が送ったコメントの右側に × ボタン** が出るので、それで削除可能

> 同じスマホ・同じブラウザで送信したコメントだけ × が出ます。別ブラウザや別端末から書いたコメントは削除できません。

### B-5. 共有する (SNS / メッセージ)

画面右下の **⤴ ボタン** で展覧会全体の URL を共有できます。

- スマホなら共有シート (Instagram / X / LINE / メール 等) が出る
- PC なら URL がクリップボードにコピーされる

---

## C. プライバシーについて

- 個人を特定できる情報 (氏名・メールアドレス等) は送信されません
- ニックネームは表示用のため、本名でなくて構いません
- いいねの重複防止のためブラウザ単位の匿名 ID が記録されますが、誰がいいねしたかは特定できません
- 匿名 ID はブラウザ内の複数の場所に分散して保存されており、ブラウザの履歴やキャッシュを少し消した程度では維持されます。「閲覧データを完全に削除」するとリセットされ、自分のコメントの × ボタンや「いいね受付済み」表示がなくなります

---

## D. うまく動かないとき

- **ページが真っ白**: 画面を下に引っ張って更新 (リフレッシュ)
- **送信に失敗する (通信エラー)**: 場所を移動して電波の良い所で再送信
- **いいねボタンが反応しない**: 主催者がいいね機能を OFF にしている可能性
- **コメント欄が出ない**: 主催者がコメント機能を OFF にしている可能性
- **「閲覧には QR コード経由のアクセスが必要です」と出る**: 招待タイプの展覧会を、招待 URL ではない普通の URL で開こうとした場合です。主催者から正しい QR / URL を再度受け取ってください
