# 主催者向けマニュアル

展覧会を主催する方向けの手順書です。展覧会の作成からキャプション印刷、Web 展覧会の公開、データ書き出しまでを順番に説明します。

> **現在のバージョン**は各画面下部のフッタに表示されています (例: `Qriine v0.18.1 (2026-06-26)`)。サポートに問い合わせる際は併せてお伝えください。リリースごとの変更点は [CHANGELOG.md](https://github.com/printrymist/exhibition-system/blob/main/CHANGELOG.md) にまとめています。

---

## 0. 用語と画面の流れ

- **展覧会コード** (例: `TE001`): 各展覧会につく識別番号です。展覧会作成時に発行されます。
- **主催者ログイン**: 各画面を開く際に必要です。メールアドレスを入力 → 届いたリンクをクリック、で完了。一度ログインすると同じブラウザでは数日間維持されます。

> 作家が使う「作品入力画面」と来場者が使う「作品ページ (QR から開く)」「Web 展覧会」はログイン不要です。

主な画面と用途:

| 画面 | 用途 |
| --- | --- |
| 展覧会作成 | 新規展覧会を申請する出発点 |
| 作品登録/設定 | 入力項目の設定・作家の招待・作品枠追加・感想受付期間・作品ロック |
| 作品入力 (作家用) | 作家が自分の作品データを入力する画面 |
| キャプション | キャプション印刷・QR コード一括印刷・テンプレート管理 |
| ダッシュボード | いいね/コメントのリアルタイム表示 |
| データ書き出し | 作品データ・コメント・いいねを Excel / CSV で取得 |
| Web 展覧会の設定 | オンラインギャラリーの公開設定・外観・会場 QR 発行 |

よく使う画面はブラウザのブックマークに登録しておくのがおすすめです。

## 1. 新しい展覧会を作る

1. 「展覧会作成」画面を開く
2. 展覧会名・会期・会場・主催者メールアドレス等を入力
3. 必要に応じて「**🧪 練習モードで試す**」にチェック (詳細は §1.5 参照)
4. 「登録」 → 申請完了
5. 入力したメールアドレス宛に確認メールが届く → リンクをクリック → セットアップ画面へ
6. セットアップ画面で展覧会コード等を入力 → Submit
7. 完了画面で発行された **展覧会コード** を控えてください (例: `TE001`)
8. 完了通知メールが届きます (各種 URL 入り)

> ここで入力した「主催者メールアドレス」が、以降この展覧会を編集できる人のアドレスになります。「データ書き出し」画面でログインする際もこのアドレスを使います。

## 1.5. 🧪 練習モード (おすすめ: 初めて使う方は必ず)

いきなり本番運用するのは不安、という方向けの練習モードです。

### 申請時に「練習モードで試す」にチェック

すると以下が変わります:

- 確認メール・完了メールに「[練習モード]」が付く
- 各画面 (作品登録/設定・キャプション・データ書き出し) の上部に **「🧪 練習モード — 残り N 日で自動削除」** バッジが表示される
- 来場者・作家の画面にも「🧪 これは練習用展覧会です」バナーが表示される
- 作成される案内文の先頭に「※これは練習モードの確認用です。実在の作家には送らないでください」と警告が入る
- 「📨 自分宛にメールで送る」を押す前にも「実在の作家へ転送禁止」確認ダイアログが出て、件名に「[練習モード - 転送禁止]」が付く
- 作家が新規登録しようとすると「これは練習用です。14 日後に削除されます」と確認ダイアログが出る

### 14 日後に自動削除

申請から 14 日経過すると自動的に削除されます。削除前日にも通知メールが届きます。

削除されるもの:
- 設定・テスト作品データ・コメント・いいね・画像ファイル、すべて消去
- 主催者の管理画面からも見えなくなります

### 🎓 本番運用に切替 (卒業)

練習で作った設定を気に入って、そのまま本番運用したい場合。練習で入力した作品・コメント・いいね・画像を **残すか消すか** を選べます:

1. 「作品登録/設定」画面の設定タブを下までスクロール (練習モードのときだけ「🎓 練習モードを終了して本番運用に切替」セクションが表示されます)
2. 2 つのボタンから選びます:
   - **🎓 練習データを残して本番化**: 練習で入れた作品・コメント・いいね・画像を **そのまま本番として引き継ぎ** ます
   - **🗑️ データを消して本番化**: 練習データを **すべて削除** してまっさらな状態で始めます (確認のチェックが必要)
3. どちらの場合も **設定 (項目設定・キャプションテンプレート等) はそのまま残り**、練習モード扱いが解除され、自動削除予定もキャンセルされます

### よくある質問

**Q. 練習中に作家へ案内を送ってしまったら?**
A. 練習モードでは案内文の先頭に「※これは練習モードの確認用です。実在の作家には送らないでください」と警告が入ります。万一そのまま送ってしまっても、作家側の入力画面にも「これは練習用です」とバナーが出るので問題は起こらない設計です (練習用の登録内容は 14 日後に自動削除)。

**Q. 練習モードで実際の QR 印刷を試したい**
A. 印刷自体は可能ですが、QR を会場に貼ると来場者が練習データに感想を書き込んでしまいます。練習モードでは QR 印刷の動作確認のみに留め、実際の貼り出しは本番運用に切替えてからにしてください。

**Q. 14 日では試す時間が足りない**
A. 一度卒業ボタンで本番運用に切り替えれば自動削除はキャンセルされます。練習用バナーや警告ダイアログも消えるため、その状態で実運用に移行する形になります。もう一度練習モードで試したい場合は、新しい練習モード展覧会を改めて作成してください。

## 2. 入力項目を設定する

1. 「作品登録/設定」画面を展覧会コードを指定して開く
2. 「主催者ログイン」カードが出るので、登録時のメールアドレスを入力 → 「リンクを送信」
3. 届いたメールのリンクをクリック → 自動でログインされ、設定画面が開きます
4. 「⚙️ 項目設定」タブで使用する項目を選択 (チェック ON / 必須マーク / ドラッグで並び替え)
5. 「💾 保存する」をクリック

> 一度ログインすれば同じブラウザでは数日間維持されます。毎回メールリンクを開く必要はありません。

### 主催者ご自身が過去に実施した展覧会から設定を複製する

- 「📥 他の展覧会から設定を複製」をクリック
- パネルが開きます。複製したい内容のチェックボックスを選択:
  - **項目設定**: 作品入力フォームに出すフィールドの一覧
  - **キャプション設定**: キャプション印刷時の項目選択 + 最後に使ったテンプレート (キャプション画面で再選択不要)
  - **🖼️ Web 公開設定**: Web 展覧会の公開状態 / いいね・コメント許可 / ヒーロー画像 / 背景画像
- 検索ボックスで展覧会名を絞り込み → 該当展覧会の「複製」
- 必要に応じて編集して「💾 保存する」

## 3. 作家を招待する

1. 「② 作家を招待」タブを開きます
2. 「作家名」欄に名前を入れるかどうかで挙動が変わります:
   - **空欄** → 全作家共通の URL になる (作家は画面で自分の名前を入力して開始)
   - **入力あり** (例: `田中太郎`) → その作家専用の URL が発行される (作家名が自動入力され、別作家の作品には触れない)
3. 「📝 案内を作成」をクリック → **招待リンク**と**案内文**(入力フォームURL＋入力項目の説明＋手順)が画面に表示されます
4. 「リンクをコピー」または「案内文をコピー」を押して、**LINE・メール・DM など、ふだん作家と連絡している方法でそのまま送ってください**
5. メールで受け取りたい場合は「📨 自分宛にメールで送る」も使えます (主催者ご自身に届くので、確認のうえ作家へお送りください)

### 作家別 URL の使い分け (推奨)

応募が確定している作家のリストがある場合は、作家ごとに 1 つずつ案内を作成するのがおすすめです:

- 作家別 URL なら作家名のスペル違い・取り違え事故が起きない
- 各作家のプライバシーが守れる (他の作家の登録状況が見えない)

応募の枠だけ先に決めて作家名が未確定の場合は共通 URL でも OK です。

### 作品枠を追加する

- 同じ画面の「作品枠を追加」で枠数を入れて「追加」
- 指定した数だけ空き作品枠が事前予約されます
- 作家が新規登録すると、空き枠の 1 つに自動で割り当てられます

## 3.5. 🔒 作品をロックする (会期直前の入力締切)

会期が近づいて「もう変更しないでください」と作家に伝えたいとき、または会期中に内容を凍結したいときに使います。ロック中の作品は、作家側の入力画面では各項目が灰色になり編集できなくなります。

### 個別ロック (作家側からの自主提出)

作家が作品入力画面で **「✓ 提出する」** ボタンを押すと、その 1 件だけがロックされます。「自分はもう編集しません」という意思表示。

### 主催者側で個別ロック / 解除

「作品登録/設定」画面 → **「📋 作品一覧」** タブ → 該当作品を展開 → 編集パネル末尾の **🔒 / 🔓 ボタン**

### 一括ロック / 解除

「⚙️ 項目設定」タブの中段「**作品の一括ロック**」セクションへ:

- **🔒 全作品をロック**: 登録済の全作品を一気にロック (会期直前に推奨)
- **🔓 全作品をロック解除**: 全作品を解除 (会期後の修正受付・トラブル対応時)

> 会期当日は一括ロックしておくと、来場者・SNS 公開後に作家が誤って画像を差し替えてしまう事故を防げます。

**必須項目チェック**: 「🔒 全作品をロック」(または作品一覧タブで個別の「🔒 ロック」) を押した時、必須に設定されている項目に空欄の作品があるとロックが中断され、どの項目・どの作品で空欄が出ているかが表示されます。対応方法は 2 通り:

- 該当作品にデータを入力してから再度ロックする
- 「項目設定」タブで該当項目の「必須/任意」を「任意」に変えてから再度ロックする (例: 価格を必須にしていたが非売の作品が混ざる場合など)

「必須」の意味を強く保ちたいので、ロックの確認ダイアログ 1 つでまとめて必須設定を解除する導線は意図的に作っていません。

## 3.7. 会場・会期中の修正対応

会期が始まってから「タイトルにタイプミスがあった」「価格を変えたい」「画像を撮り直して差し替えたい」といった修正が必要になることがあります。スマホやノート PC から会場でも対応できます。

### A. 主催者が直接修正する (もっとも素早い)

1. スマホかノート PC で「作品登録/設定」画面を開く
2. 主催者ログイン (一度ログイン済みなら数日間維持されます)
3. 「📋 作品一覧」タブで該当作品を探す
4. 行をタップして展開 → 内容を直接編集 → 「💾 更新する」

> 作品がロック中でも主催者には編集制限はかかりません。「ロック中だから直せない」と心配せず編集して大丈夫です。

### B. 作家本人に修正してもらう

作品の差し替えなど大きな修正の場合は、作家自身に直してもらう方が手戻りが少ないことがあります。

1. 該当作品がロック中なら、まず編集パネルから 🔓 解除する
2. 作家用 URL を改めて伝える (「② 作家を招待」で案内を作り直して送ってもよい)
3. 作家が入力画面から修正
4. 修正完了の連絡を受けたら、必要に応じて再びロックする

### C. 画像だけ差し替える (会場で撮り直し)

会期前に作品写真が間に合わなかった場合や、撮り直したい場合に便利です。

1. 主催者がスマホで「作品登録/設定」画面を開く
2. 「📋 作品一覧」 → 該当作品の編集パネル
3. 画像欄の「ファイルを選択」 → スマホで撮影、またはアルバムから選択
4. 自動で長辺 1600px に縮小されるので、原寸そのままでも OK
5. 「💾 更新する」

### D. 作家本人が自分のスマホから写真を更新する (ショートカット)

会場で作家本人がいる場合、作家用 URL や入力画面を開かずに、**来場者用 QR からそのまま写真を更新** できる裏口があります。複数作品の写真追加・差し替えを素早く済ませたいときに便利です。

1. 作家が作品の横の **QR コード** を読み取る (来場者用の作品ページが開く)
2. 感想入力フォームの **「ニックネーム」欄に作家名 (登録時と同じ綴り)** を入力
3. 「🎨 作家本人として認識: ○○○○」のメッセージとともに **「📷 写真を選択して更新」** ボタンが現れる
4. 作家が写真を選択 → 自動アップロード

文字情報は変更されず、画像だけ上書きされます。

> ロック中の作品はこの方法でも写真更新できません。事前にロック解除してください。
> 作家には「作品の QR を読んで、ニックネーム欄に自分の作家名を入れると写真が更新できる」と一言伝えれば OK です。詳細は作家向けマニュアル §7.3 にも書いてあります。

### よくある状況

**Q. 値札 (キャプション) を会期途中で刷り直したい**
A. 「キャプション」画面で印刷 → ブラウザの印刷ダイアログで該当ページだけ指定するのが手っ取り早いです。

**Q. 全部の作品を会期中に編集禁止にしたい**
A. §3.5 の「🔒 全作品をロック」を使ってください。誤操作でデータが壊れる事故を防げます。

**Q. ロック中でも来場者のいいね・コメントは普通に集まる?**
A. はい。ロックは作家の編集を止めるだけで、来場者のいいね・コメントには影響しません。

## 4. 感想 (コメント) 受付期間の設定

1. 「⚙️ 項目設定」タブの下方「感想 (コメント) 受付期間」へ
2. 開始/終了の日時を入力 (空欄なら常時受付)
3. 「保存」
4. 来場者の感想画面では、期間外になると入力欄の代わりに「感想の受付期間は終了しました。」(または開始前なら開始日時のお知らせ) が表示されます

## 5. キャプション/QR を印刷する

### 5.1. 画面を開く

1. 「キャプション」画面を展覧会コードを指定して開く
2. 「主催者ログイン」カードが出ている場合は主催者ログイン (§2 と同じ手順)。ログイン済みなら直接展覧会データが読み込まれます

### 5.2. 推奨フロー (4 ステップ)

「🖨️ キャプション印刷」画面最上部の「💡 おすすめの使い方」にも書いてありますが、迷ったらこの順序が最短です。設定タブは使う順に **① テンプレート → ② 載せる項目 → ③ 詳細編集 → ④ ページ設定** と番号が付いています:

1. **① テンプレート** で目的に近い **⭐ 公式テンプレ**を選んで **📥 採用** (カードのクリックはプレビューだけ。採用で確定)
2. **② 載せる項目** でキャプションに載せる項目を確認・調整
3. (必要なときだけ) **③ 詳細編集** で並び・大きさ、**④ ページ設定** で用紙・余白・QR を調整
4. **「この展覧会のテンプレとして保存」** — 別テンプレを試す前のセーフネットになる (③詳細編集タブのほか、④ページ設定タブ下部のボタンからも保存できます)

### 5.3. テンプレ一覧

「① テンプレート」タブには 13 個のテンプレが並びます。⭐ マーク付きが**公式推奨** (9 個)、それ以外がベース (4 個):

| | 名前 | 用途 | 採用時に Page 設定も適用 |
|---|---|---|---|
| ⭐ | 🎨 個展 推奨 | 1 人の作家、ゆったり展示 (A7・2×4) | ○ |
| ⭐ | 🎭 グループ展 推奨 | 多人数展、コンパクト (3×5) | ○ |
| ⭐ | 📷 写真展 推奨 | 撮影地・撮影年・カメラ項目 | ○ |
| ⭐ | 🌐 国際展 推奨 | 日英併記、中央寄せ | ○ |
| ⭐ | 🎏 シンプル・中央 推奨 | 全項目センタリング・日本のギャラリー定番 | ○ |
| ⭐ | 💬 作家コメント入り 推奨 | 作家コメント (artist_note) を下段に添える | ○ |
| ⭐ | 📌 展覧会名ヘッダー 推奨 | 上端に展覧会名の帯 (グレー背景・自動投入) | ○ |
| ⭐ | 🎨 版画・限定版 推奨 | エディション・額装価格表示。版画・限定版作品向け | ○ |
| ⭐ | 📍 QR のみ (NY スタイル) | 紙には QR シールだけ、スマホで詳細表示。NY 系ギャラリーの「作品優先・説明排除」スタイル / 印刷コスト最小化 | ○ |
| | ✂️ シンプル | タイトルと作家名のみ | — |
| | 🖼️ ギャラリー | QR 左上 / 画廊風 | — |
| | 🇯🇵 作品名先 | 和風、作品名から始まる | — |
| | 🇯🇵 作家名先 | 和風、作家名から始まる | — |

> 📍 **QR のみ (NY スタイル)** は、④ ページ設定タブの **紙の出力モード**を「QR シールのみ印刷 (情報はスマホ表示)」に設定するテンプレです。この出力モードは全テンプレ共通の設定なので、他の公式テンプレ (例: 個展 推奨) を採用したあと出力モードだけ「QR シールのみ」に切替えれば、好みのレイアウトで QR シール運用に転用できます。

**採用時の挙動 (重要):**

| 層 | 挙動 |
|---|---|
| 作品登録の項目設定 (registration_fields) | **追加** — テンプレが必要とする項目で不足分だけ append、既存項目はそのまま残る |
| ② 載せる項目 (caption_fields) | **書き換え** — テンプレの並び順・サイズ等で完全置換 |
| ④ ページ設定 (列数/行数/QR サイズ/余白 等) | 公式テンプレのみ**書き換え**、それ以外のテンプレは現状維持 |

**カードの見た目:**
- ⭐ + ゴールド枠 = 公式推奨
- 緑グラデ + ✅ 採用中 表記 = いまこの展覧会で採用中のテンプレ
- 「📥 採用」ボタン (薄い青) を押すと採用ダイアログが出ます

### 5.4. 詳細な編集とページ設定

採用後の細部調整は以下のタブで行います:

- **② 載せる項目**: 作品登録で集めた項目のうち、キャプションに載せる項目を選びます (チェックを入れた項目がキャプションに使われます)
- **③ 詳細編集**: キャプションに出す項目の並び順、フォントサイズ、太字、最大行数 (1〜3 行・制限なし) など。作家コメント等の**改行はそのまま印字**されます (行数が多いときは最大行数を上げてください)
- **④ ページ設定**: 用紙サイズ (A4 / Letter)、向き、列数・行数、QR 位置・サイズ、余白、フォントファミリ、**テキスト色** (黒 / 濃グレー / 中グレー)、固定文 (展覧会名ヘッダー等)

**QR の位置**は「左上 / 右上 / 左下 / 右下 / 上中央 / 下中央」の固定配置のほか、**文字を QR で折り返す**設定 (左上・右上・**左下・右下**に折り返し) も選べます。折り返しを使うと QR の周りに文字が回り込み、スペースを有効に使えます。

**🖨 カードシート印刷 (エーワン等の型抜きカード)**: ④ページ設定タブの「カードシート印刷」を **ON** にすると、カードの幅・高さ・上下左右の余白・カード間隔 (mm) を指定して、型抜きカードに合わせて印刷できます (使うときだけ ON。OFF なら従来どおりの印刷です)。初期値は A4 の名刺 10 面 (面付けは「列数 2・行数 5」)。テスト印刷して位置がズレたら、上余白・左余白・間隔を 0.5mm 単位で微調整してください。カードシート印刷のときはキャプションの枠線は自動で消えます。

**保存**: 満足したら **「この展覧会のテンプレとして保存」** で保存します (③詳細編集タブの保存欄、または④ページ設定タブ下部のボタン)。
- 展覧会ごとに保存され、別名で複数保存できます (用紙・カードシート等のページ設定も一緒に保存されます。保存しないと次回開いたときに戻ります)
- 過去の展覧会のテンプレを読み込んで再利用することもできます

### 5.5. QR コードの一括印刷

画面上部の **「🔲 QRコード印刷」** タブ (キャプション印刷タブの隣) で、QR コードだけを別途一括印刷できます。会場での貼り出しや、QR シールを作品の裏に貼る用途など。

## 5.6. 🌐 Web 展覧会を公開する

会場に来られなかった人向けや、会期後のアーカイブ閲覧用に、作品をオンラインで一覧できる Web 展覧会を公開できます。

### 5.6.1. Web 展覧会の設定画面を開く

主催者専用 URL: `https://qriine.com/web-exhibition.html?ex=○○○`
(`○○○` は展覧会コード)

主催者メールアドレスでログインしてください。

### 5.6.2. 公開設定

「公開設定」カードで 2 つを選びます。

**公開状態**:
| 選択肢 | 動作 |
|---|---|
| 🔒 **非公開** | 誰も見られません (準備中・休止中はこれ) |
| 🎫 **QR を持つ来場者のみ** | 主催者が発行した会場 QR / リンクを持つ人だけ閲覧可 (会期中の招待制運用) |
| 🌍 **公開** | URL を知っていれば誰でも見られます (SNS 拡散・アーカイブ運用) |

**Web 展覧会で来場者ができること**:
| 選択肢 | 動作 |
|---|---|
| **閲覧のみ** | 作品を見るだけ。いいね・コメントの機能はなし |
| **いいねのみ** | ハートボタンでいいねできる |
| **いいね + コメント** | いいね + 💬 でコメントも書ける |

「💾 設定を保存」で反映されます。

### 5.6.3. ギャラリー外観

「🎨 ギャラリー外観」カードで Web 展覧会の見た目をカスタマイズできます:

- **タイトル / サブタイトル**: SNS でシェアしたときのプレビューにも使われます
- **ヒーロー画像**: ページ上部の大きな画像 + SNS シェア時のプレビュー画像 (公開する展覧会では**設定推奨**)
- **背景画像 / 背景色**: 雰囲気作り (背景画像があればそちらが優先)

「💾 外観を保存」で反映されます。

### 5.6.4. 会場 QR コードの発行 (招待制モード用)

公開状態が「QR を持つ来場者のみ」のときに使う、**有効期限付きの専用 URL** を発行する機能です。会場入口に QR を貼っておけば、来場者がスマホで読み取って Web 展覧会を見られるようになります。期限が切れると無効になるので、会期終了後に意図せずアクセスされる心配がありません。

1. 「🎫 会場 QR コードの発行」カードへ
2. 有効期限 (日数) を入力 (デフォルト 60 日)
3. 「QR を発行」 → URL と QR 画像が表示される
4. ダウンロード → 印刷して会場に貼る、または DM 等で配布

> 公開状態が「公開」モードならこの操作は不要です。Web 展覧会の URL を SNS や Web サイトに貼るだけで誰でも見られます。

### 5.6.5. SNS で告知する

「公開」モードに切り替えた後の SNS 告知方法は **SNS 告知マニュアル** を参照してください。

## 6. 出力・レポート (ハブ画面)

展覧会のリアルタイム表示・データダウンロード・印刷などへの入口を 1 つにまとめた画面です。「作品登録/設定」画面・「キャプション」画面のヘッダにある **「📊 出力・レポートへ」** から開けます (展覧会コードと認証は引き継がれます)。

ハブには次の行き先があります: **📈 リアルタイムダッシュボード / 📊 詳細分析 / 📥 データダウンロード / 📋 作品リスト / 🌐 Web 展覧会 / 🕐 変更履歴**。

### 📈 リアルタイムダッシュボード

- 参加者数 / いいね数 / コメント数 / 登録作品数の大きな数字
- **いいねランキング 3 種**:
  - 🏆 **総合**: 会場 QR + Web 展覧会 の合算
  - 🏛️ **会場**: 会場 QR からのいいねのみ
  - 🌐 **Web**: Web 展覧会からのいいねのみ
- 最新コメントフィード
- 会場控え室モニタ等に常時表示する用途
- 「→ 開く (別タブ)」をクリックすると新しいタブで開きます

> 📌 ハブ画面・データダウンロード画面と同様に、ダッシュボードも**主催者本人 (展覧会作成時に登録したメールアドレス) でログインしていないと閲覧できません**。未ログイン時は「🔒 運営者ログインが必要です」と表示されます。別のタブ等で既にログインしていればそのままセッションが引き継がれます (= 会場で 1 度ログインしてしまえば、当日中はずっと表示しっぱなしで OK)。

### 📊 詳細分析

来場者の関与パターンを会期中・会期後に振り返るための画面です。コメント率・関与の段階・滞在時間の中央値・リピート率・ロングテール度、作家別リーチ (グループ展) などを表示します。「→ 開く」から開きます。

### 📥 データダウンロード

「→ 開く」をクリックすると、展覧会選択済みの状態で開きます。

ダウンロードできる種類:

- 🎨 **Artworks XLSX**: 作品データ。Excel で開いても型崩れしません (エディション「5/30」が日付に化けない・価格や品番の書式も保持されます)
- 💬 **Comments CSV**: コメント全件 (時刻順)
- 👍 **Likes CSV**: いいね一覧
- 📊 **Summary CSV**: 作品別の like/コメント数集計

> ハブ画面・ダウンロード画面ともに、主催者本人 (展覧会作成時に登録したメールアドレス) でログインしないと閲覧できません。展覧会作成時のアドレスとログインアドレスが一致していないと「展覧会がありません」と表示されます。

### 📤 XLSX のアップロード (Import)

ダウンロードした **Artworks XLSX** を Excel で編集 → 同じ画面の「📤 Import」セクションでアップロードすると、編集内容が Firestore に反映されます。「Excel で一気に修正したい」「PC でまとめて入力したい」主催者向けの機能。

**XLSX の構造 (ダウンロード時)**

- 登録済の作品 (status='1') と、まだ埋まっていない空きスロット (status='0') を**両方含む**
- 列は「作品情報」(タイトル・年・技法・サイズ・価格 等、項目設定で ON にしたフィールド) + 「作家情報」(英語作家名・生年・出身地・SNS リンク 等)
- artwork_id 列にはあらかじめ各 slot の ID が入っている — **この列は編集しないでください**

**使い方**

1. 「🎨 Artworks XLSX」でダウンロード → Excel で開く
2. **artwork_id がある行に**作品情報・作家情報を書き込む / 修正する (空きスロットの行を埋めることで新規登録ができます)
3. 「📤 Import」でファイルを選択 → 「🔍 内容を確認」
4. 「編集 / 新規登録 / 触らない」の件数プレビューが出る (エラーがあれば修正点が一覧表示される)
5. 問題なければ「✅ この内容で実行」で書き込み

**ルール (動作)**

| | 挙動 |
|---|---|
| 登録済の作品の行を編集 | 該当作品を**更新** |
| 空きスロットの行にタイトル等を入力 | **新規登録** (空きスロットから登録済に格上げ) |
| 空きスロットの行を空のまま | 何もしない (空きスロットのまま残る) |
| xlsx の値が既存と全く同じ行 | 書き込みスキップ (変更履歴にも残らない) |
| 登録済の行のタイトルを空にする | **拒否** (誤削除と思われるため。本当に消したい時は「作品登録/設定」画面から操作してください) |
| xlsx から行を削除 | 作品データは触らない (誤削除事故防止) |
| 同じ作家名の複数行で作家情報の値が違う | **拒否** (1 つに揃えてから再アップロード)。空セルは「他行から継承」として扱われるので、新規登録行で作家情報の再入力は不要 |
| ロック中の作品の行 | **拒否** (「作品登録/設定」画面でロック解除してから) |
| 作品 ID を空にする / 変える / 重複 | **拒否** (作品 ID 列は触らない設計) |

**作品枠を増やしたい場合**

アップロード経由ではなく、「作品登録/設定」画面の「項目設定」タブ末尾「作品枠を追加」で先に枠を作ってから、再度ダウンロードしてください。新しい空きスロット行が xlsx に現れます。

**画像について**

作品画像 (image_url) は xlsx には含まれません。画像の登録・差し替えは「作品登録/設定」画面から行ってください。新規登録した作品の画像も同様です。

エラーが 1 つでもあれば**一切書き込まない**設計なので、「うっかり大量上書き」の事故は防げます。

### 📋 作品リスト

出展作品の一覧表を作って **紙 / PDF に印刷** できます (価格表・作品目録・受付台帳など)。「→ 開く」から専用画面が開きます。

- **名前を付けたリストを複数**作れます。1 つの展覧会で「価格表」「作品目録」「受付台帳」のように、**載せる項目やレイアウトの違う表を別々に保存**できます。上部の「リスト」で切り替え、**＋新規 / 複製 / 削除**で管理します
- **リストのタイトル**、**表示する項目と列の順番**、**見出しの配置** (左寄せ / 中央) を選べます
- **列の幅**を調整できます。「📐 データから列幅を自動調整」を押すと、登録済みの作品の中身から各列にちょうどよい幅 (比率) を自動で割り当てます。各列の「幅」欄で手直しもできます (空欄なら均等)
- **作品の並び順**をドラッグで自由に指定できます (作品リスト印刷用。作品 ID は変わりません)。並び順はこの展覧会で 1 つを全リスト共通で使います
- **用紙** (A4 / Letter)、**向き** (縦 / 横)、**段組** (縦は最大 2 段 / 横は最大 3 段) に対応。1 ページにより多くの作品を載せられます。段組や作品が多いときも、**自動でページに分かれて続き**、各段の先頭に見出しが付きます
- 設定 (各リスト・並び順) は展覧会ごとに保存されます

> キャプション (作品ごとのラベル) とは別の、全作品を表にした一覧です。
>
> 印刷時のヒント: 段組で行が次のページに少しはみ出す・余白が大きすぎるときは、プリンタ側の「拡大縮小」を 100%、余白を「標準」にすると安定します。

### 🕐 変更履歴 (Audit Log)

作品データに対する変更を時系列で確認できます。「出力・レポート」画面の「🕐 変更履歴」カードから開いてください。

**表示内容** (直近 100 件まで時系列降順):

- いつ (時刻)
- 誰が (主催者メール / 運営者 / 作家 / 匿名トークン)
- どの作品 (artwork_id)
- どのフィールドを変えたか (タイトル / 価格 / 技法 …)
- 行を展開すると **before → after** の差分が表示されます

**使いどころ:**

- 「会期前に価格を一斉に変えたつもりだが、どの作品を変えたっけ?」を後追いで確認
- 作家が「私の作品が消えました」と連絡してきた時、誰がいつ削除/編集したかを追跡
- XLSX 一括反映の影響範囲を後で見返す

ロック状態の変更 (`is_locked` の true ⇔ false) も記録されるので、誰が解除したかも追えます。

## 7. トラブルシューティング

- **ログインリンクのメールが届かない**: 迷惑メールフォルダを確認してください。Gmail なら「迷惑メールではない」を一度押すと、以降は受信トレイに届きます。
- **ログインリンクをクリックしたら「期限切れ」「無効」と出る**: リンクには有効期限 (約 1 時間) があります。もう一度メールリンクを送信してください。Outlook 等が先回りでリンクをプレビュー取得してしまうケースもあるので、メールアプリのリンクプレビューを切るか、別のメーラーで開くと回避できます。
- **「展覧会がありません」と表示される (データ書き出し画面)**: ログインしたメールアドレスと、展覧会作成時に登録した主催者メールアドレスが一致していない可能性があります。
- **テンプレートを Load しても項目が出ない**: 現在の展覧会の「項目設定」で当該項目が ON になっていない可能性。項目設定タブで確認してください。
- **画面更新が反映されない**: ブラウザキャッシュが古い可能性。`Ctrl+Shift+R` (Mac: `Cmd+Shift+R`) で強制再読み込みしてください。
- **作品枠を増やしたい**: 「項目設定」タブ末尾「作品枠を追加」
- **過去展覧会の設定を全部使いたい**: 項目設定タブの「📥 他の展覧会から設定を複製」で「項目設定」「キャプション設定」「Web 公開設定」のチェックボックスにすべてチェックを入れて複製してください。キャプションテンプレートの選択もそのまま引き継がれるので、キャプション画面で再選択する必要はありません。
- **作家から「ロックされていて編集できない」と連絡が来た**: 個別解除なら「作品一覧」タブから該当作品を展開して 🔓 ボタン、全件解除なら「項目設定」タブの「🔓 全作品をロック解除」
- **Web 展覧会の URL を SNS に貼ってもプレビュー画像が出ない**: Web 展覧会の設定画面 → ギャラリー外観 で **ヒーロー画像** を設定してください。設定後は SNS 側のキャッシュで古いまま見える場合があるので、SNS 告知マニュアル §4「プレビュー画像の確認と更新」を参照。
- **Web 展覧会で「閲覧には QR コード経由のアクセスが必要です」と表示される**: 公開状態が「QR を持つ来場者のみ」のままです。SNS 拡散したいなら「公開」に切り替えてください。
