Recover Exhibition (ex_code 指定で Firestore に再書き込み)

運営者管理者ログインが必要です
この管理ツールは Firestore への書き込みのため、運営者管理者 (rymist1@gmail.com) のみ実行できます。
用途: GAS 認証 (Email Link) は通ったが Firestore exhibitions/{exCode} への書き込みが取りこぼされた展覧会を、Master SS の値から再構築して書き戻します。
前提: Master SS の exhibitions シートに該当 ex_code の行が存在すること。
動作: Cloud Function adminRecoverExhibitionDoc が GAS から canonical doc を取得し、set(..., { merge: true }) で Firestore に書き込みます (冪等)。
